Textie Editor

UWP free source code and text editor app.

Features

  • Tabbed editing
  • Open and Save
  • Auto-save, Save as and Save all
  • Undo and Redo
  • Find and Replace
  • Search text with search engines
  • Print, Share, Pin file to Start menu
  • Hot Exit for Unsaved Files
  • Recent Files
  • Modern UI and comfortable UX
